
# city-crud-ui

This is the frontend UI for the City CRUD project.

## Features
- React-based user interface
- Add, view, update, and delete cities
- Connects to the backend API
- Modern UI with Vite

## Prerequisites
- Node.js (v16 or higher recommended)
- npm

## Setup & Run
1. Open a terminal in the `city-crud-ui` directory.
2. Install dependencies:
	```sh
	npm install
	```
3. Start the development server:
	```sh
	npm run dev
	```
	The app will run on `http://localhost:5173` by default.

## Project Structure
- `src/` - React components and styles
- `public/` - Static assets

## Notes
- Make sure the backend API is running for full functionality.
- You can configure the API endpoint in the frontend code if needed.
